Produtos da compra
Informe nome, quantidade e preço unitário de cada item.
Consultas de Banco de Dados
Estruturas SQL resolvidas para o cenário proposto.
Questão 1 - Valor total de cada compra
SELECT 
    c.ID_Compra,
    p.Nome_Produto,
    c.Quantidade,
    p.Preco_Unitario,
    (c.Quantidade * p.Preco_Unitario) AS Valor_Total_Compra
FROM 
    Compras c
INNER JOIN 
    Produtos p ON c.ID_Produto = p.ID_Produto;
Questão 2 - Produto mais vendido
SELECT 
    p.ID_Produto,
    p.Nome_Produto,
    SUM(c.Quantidade) AS Quantidade_Total_Comprada
FROM 
    Compras c
INNER JOIN 
    Produtos p ON c.ID_Produto = p.ID_Produto
GROUP BY 
    p.ID_Produto, 
    p.Nome_Produto
ORDER BY 
    Quantidade_Total_Comprada DESC
LIMIT 1;
Questão 3 - Todos os produtos e totais
SELECT 
    p.ID_Produto,
    p.Nome_Produto,
    COALESCE(SUM(c.Quantidade), 0) AS Quantidade_Total_Comprada
FROM 
    Produtos p
LEFT JOIN 
    Compras c ON p.ID_Produto = c.ID_Produto
GROUP BY 
    p.ID_Produto, 
    p.Nome_Produto
ORDER BY 
    p.ID_Produto ASC;
